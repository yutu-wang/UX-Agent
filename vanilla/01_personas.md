# T-Pass 使用者人物誌（Personas）

---

## Persona 1 — 陳大偉（通勤族）

| 欄位 | 內容 |
|------|------|
| 年齡 | 32 歲 |
| 職業 | 軟體工程師 |
| 居住 | 新北市板橋 → 台北市信義區 |
| 搭乘頻率 | 每日 2 趟，週一至週五 |
| 裝置 | iPhone，習慣用 Apple Pay |

**目標**
- 確認每月折扣有正確套用
- 不想手動儲值，希望自動扣款
- 快速查看本月省了多少

**痛點**
- 不確定 T-Pass 折扣是否生效
- App 通知太少，無法得知本月搭乘進度
- 換信用卡後重新綁定流程繁瑣

**引述**
> 「我只想知道這個月省了多少錢，但 App 裡找很久才找到。」

## Persona 2 — 林怡君（大學生）

| 欄位 | 內容 |
|------|------|
| 年齡 | 20 歲 |
| 身份 | 大學二年級，通學生 |
| 居住 | 台中市北屯區 |
| 搭乘頻率 | 每週 5 天，搭公車上下學 |
| 裝置 | Android，習慣 Line Pay |

**目標**
- 取得學生優惠資格
- 全程用手機完成，不想跑門市
- 清楚知道哪些路線適用

**痛點**
- 學生資格驗證流程不透明、卡關
- 不清楚台中哪些公車在 T-Pass 範圍
- 審核結果通知不顯眼

**引述**
> 「我以為申辦很簡單，結果卡在學生認證那邊超久。」
