# 產品

名稱：TPASS Redesign
一句話：重新設計 TPASS 官網與服務流程，降低使用者理解門檻。
目標：

* 幫助使用者快速理解 1.0 / 2.0 差異
* 降低方案選擇焦慮
* 提升資訊清晰度與操作效率

# 研究風格

方法：混合（訪談＋問卷＋觀察）
關注：

* 使用者困惑與決策心理
* 通勤行為與使用情境
* 對回饋機制的理解程度

# 設計風格

調性：極簡、專業、親切
品牌色：

* TPASS Green #3CB371
* Deep Gray #2B2B2B
* Light Gray #F5F5F5

避免：

* 過度漸層
* 過多 emoji
* 複雜裝飾與科技感

# 評估重點

最在意：

* 學習曲線
* 資訊理解效率
* 決策清晰度

次要：

* 情感體驗
* 公共服務信任感
